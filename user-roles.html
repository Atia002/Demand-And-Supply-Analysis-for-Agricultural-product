User Roles and Permissions

System Administrator
Access Level: Full System Access
Responsibilities:
  • System maintenance and monitoring
  • User account management
  • Data verification and validation
  • Report generation and analysis
  • Market trend analysis
Permissions:
  • Full CRUD operations on all data
  • User management
  • System configuration
  • Analytics access
  • Report generation
Farmer
Access Level: Farm Management
Features:
  • Farm profile management
  • Production record keeping
  • Weather data access
  • Market price monitoring
  • Storage facility access
Permissions:
  • Manage own farm data
  • View market prices
  • Update production records
  • Access weather forecasts
  • Contact vendors
Vendors (Wholesalers & Retailers)
Wholesaler
  • Bulk purchase management
  • Minimum order quantity setting
  • Price negotiations
  • Storage facility access
  • Market trend analysis
Retailer
  • Store inventory management
  • Consumer price setting
  • Order management
  • Consumer feedback access
  • Local market analysis
Consumer
Access Level: Basic
Features:
  • Product browsing
  • Price comparison
  • Feedback submission
  • Market information access
  • Order history tracking
Permissions:
  • View product information
  • Submit feedback
  • Update profile
  • Contact vendors
  • View public market data
